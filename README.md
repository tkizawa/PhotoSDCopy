# PhotoSDCopy (C# WPF版)

SDカード内の画像や動画を日付と拡張子ごとに整理しながら、指定したフォルダへ自動的にコピーするツールです。

## 主な機能
- **自動整理**: ファイルを「日付ごと」および「拡張子ごと」にフォルダ分けしてコピーします。
- **メタデータ抽出**: 画像のEXIF情報や、動画ファイルの作成日時などのメタデータを読み取って正確な撮影日時・作成日時を判定します。
- **特定ファイルの除外**: `.xml` や `.ctg` などの不要な管理ファイルは自動的にコピー対象から除外されます。
- **GUIで簡単操作**: 直感的なUIから、SDカードのドライブとコピー先のフォルダを選択して実行できます。

## 動作要件
- **.NET 10 SDK**: 本アプリケーションは C# と WPF (Windows Presentation Foundation) で構築されており、実行およびビルドには .NET 10 SDK が必要です。

## 実行方法

本プロジェクトはコマンドライン（ターミナルやコマンドプロンプト）から直接実行できます。

1. コマンドプロンプトまたはターミナルを開きます。
2. このフォルダ（`PhotoSDCopy` フォルダ）に移動します。
3. 以下のコマンドを実行してアプリケーションを起動します。

```bash
dotnet run
```

※ 初回実行時は必要なパッケージ（`MetadataExtractor` など）が自動的に復元・ダウンロードされます。

## 操作手順
1. アプリケーションが起動したら、「コピー元 (SDカードのドライブ)」の「参照」ボタンをクリックし、読み込みたいSDカードのドライブを選択します。
   （通常はSDカードのドライブそのものか、`DCIM` フォルダを指定します）
2. 「コピー先フォルダ」の「参照」ボタンをクリックし、ファイルを保存したい場所を選択します。
3. 「実行」ボタンをクリックするとコピー処理が開始されます。画面上の進捗バーとログテキストで進行状況を確認できます。
